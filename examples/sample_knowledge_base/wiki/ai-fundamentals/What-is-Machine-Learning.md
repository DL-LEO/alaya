# What is Machine Learning?

Machine Learning (ML) is a subset of artificial intelligence that enables systems to learn and improve from experience without being explicitly programmed.

## Core Idea

Instead of writing explicit rules for every situation, ML algorithms build a mathematical model from training data. The model learns patterns and makes predictions or decisions based on new, unseen data.

## Three Main Types

| Type | Description | Example |
|:--|:--|:--|
| Supervised Learning | Learn from labeled data | Image classification |
| Unsupervised Learning | Find patterns in unlabeled data | Customer segmentation |
| Reinforcement Learning | Learn through trial and error | Game playing |

## Key Components

- **Data**: The raw material for learning
- **Features**: Relevant attributes extracted from data
- **Model**: The mathematical representation learned from data
- **Loss Function**: Measures how wrong the model's predictions are
- **Optimization**: The process of minimizing the loss function
