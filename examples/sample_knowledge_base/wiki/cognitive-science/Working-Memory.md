# 工作记忆

> 工作记忆（Working Memory）是认知心理学中用于描述"在复杂认知任务中临时存储和操纵信息"的核心概念。与短期记忆不同，工作记忆不仅是信息的被动容器，更是一个主动处理的"心理工作台"。

## Baddeley 工作记忆模型

Alan Baddeley 和 Graham Hitch（1974）提出的多成分工作记忆模型是认知心理学最具影响力的理论框架之一，其后经过多次修订（Baddeley, 2000），引入了第四个成分。

### 模型总览

```
                           ┌─────────────────────┐
                           │    中央执行器        │
                           │ (Central Executive)  │
                           │ 注意力控制 & 协调     │
                           └──────┬──────┬───────┘
                                  │      │
                    ┌─────────────┘      └─────────────┐
                    ↓                                  ↓
        ┌──────────────────────┐        ┌──────────────────────┐
        │     语音回路          │        │    视空间画板         │
        │ (Phonological Loop)  │        │ (Visuospatial Sketchpad)│
        │                      │        │                      │
        │  语音存储 (2s 衰减)    │        │  视觉缓存             │
        │  发音复述机制          │        │  空间复述             │
        └──────────────────────┘        └──────────────────────┘
                    │                                  │
                    └─────────────┬────────────────────┘
                                  ↓
                    ┌──────────────────────┐
                    │    情景缓冲器         │
                    │ (Episodic Buffer)     │
                    │  ↔ 长时记忆的接口     │
                    └──────────────────────┘
```

### 1. 中央执行器（Central Executive）

中央执行器是工作记忆系统的**核心控制组件**，负责注意力资源的分配和协调，本身不具备存储能力。

| 功能 | 说明 |
|:--|:--|
| **注意聚焦** | 将注意力投放到特定任务或信息上 |
| **任务切换** | 在多个并行任务之间灵活转移注意力 |
| **分时共享** | 同时处理多个任务时的资源分配 |
| **抑制控制** | 主动抑制无关信息的干扰 |
| **与长时记忆接口** | 触发长时记忆中的信息检索 |

**神经基础**：主要依赖前额叶皮层（PFC）和前扣带皮层（ACC）。中央执行器的损伤会导致注意力涣散、任务切换困难、无法抑制干扰——这在 ADHD 患者中表现尤为明显。

**容量限制**：中央执行器的处理能力有限，当多个任务同时竞争注意力资源时，各任务的表现会显著下降。

### 2. 语音回路（Phonological Loop）

专门处理**言语和听觉信息**的子系统，包含两个层次：

**语音存储（Phonological Store）**：
- 保持言语信息约 2 秒，之后自动衰减
- 受语音相似性效应影响——发音相似的词（如"man, cat, map"）比发音不同的词更难记忆

**发音复述机制（Articulatory Rehearsal）**：
- 通过默读或出声复述来刷新存储中的信息
- 受词长效应影响——短词比长词更容易在复述周期内完成刷新
- 发音抑制（articulatory suppression）——当人同时进行无关的口头活动时，复述机制被占用，记忆显著下降

**容量指标**：经典实验发现，大多数人能在语音回路中保持约 **7±2 个信息块**（Miller, 1956），但 Baddeley 的后续工作表明，实际容量受复述速度和词长的限制，约为 2 秒内能说完的单词数。

### 3. 视空间画板（Visuospatial Sketchpad）

专门处理**视觉和空间信息**的子系统，又分为两个独立的成分：

- **视觉缓存（Visual Cache）**：处理物体外观、颜色、形状等视觉特征
- **内部复述（Inner Scribe）**：处理空间位置和运动信息，负责主动刷新

**双分离证据**：神经心理学发现，视觉信息和空间信息由不同脑区处理——颞下回处理视觉特征，顶叶处理空间位置。某些脑损伤患者可能保留了视觉识别能力但丧失了空间导航能力，反之亦然。

**与语音回路的关系**：两个子系统彼此独立，可以并行工作——一个人可以同时默念一串数字（语音回路）并想象一幅地图的路线（视空间画板），只要中央执行器有足够的剩余资源协调两者。

### 4. 情景缓冲器（Episodic Buffer）

Baddeley（2000）在原三成分模型基础上新增的第四个成分，用于解决原模型无法解释的问题——语音回路和视空间画板的信息如何整合，以及工作记忆如何与长时记忆交互。

| 特性 | 说明 |
|:--|:--|
| **多模态整合** | 将语音、视觉、空间信息绑定为统一的情景表征 |
| **容量有限** | 约 4 个信息块（chunks），比语音回路更严格 |
| **与长时记忆双向连接** | 可以从长时记忆中提取已有情景，也可以将新整合的信息编码进长时记忆 |
| **意识通道** | Baddeley 认为情景缓冲器的内容是我们"有意识体验"的现场 |

## 工作记忆与注意力的关系

工作记忆和注意力在认知系统中紧密耦合：

1. **中央执行器本质上是注意力控制器** — 它的核心功能（聚焦、切换、抑制）正是注意力的三大基本操作
2. **工作记忆的内容决定了注意力的优先方向** — 当前"在心里盘算"的东西会自动成为注意力的优先目标
3. **注意力选择决定了工作记忆中的内容** — 你注意到什么，什么就进入了工作记忆的缓存

这与佛教唯识学中"作意（注意力唤起）→ 触（接触）→ 受（感受）→ 想（概念化）→ 思（行动驱动）"的认知流程形成有趣的呼应。
